## How to Run

1. Clone this repository.
2. Open terminal in project directory.
3. Compile:
    javac AdvancedQuizApp.java
4. Run:
    java AdvancedQuizApp

5. Questions are loaded from `questions.txt`
6. Scores are saved to `scores.txt`

## File Format for `questions.txt`

Each line:
Question Text|Option1|Option2|Option3|Option4|CorrectOptionIndex
